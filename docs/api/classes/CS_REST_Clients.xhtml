\CS_REST_Clients

Class to access a clients resources from the create send API.

This class includes functions to create and edit clients,
along with accessing lists of client specific resources e.g campaigns

  • Author: tobyb

  • CS_REST_Clients() — Constructor.
  • create() — Creates a new client based on the provided information
  • delete() — Deletes an existing client from the system
  • get() — Gets all templates the current client has access to
  • get_campaigns() — Gets a list of sent campaigns for the current client
  • get_drafts() — Gets a list of draft campaigns for the current client
  • get_lists() — Gets all subscriber lists the current client has created
  • get_lists_for_email() — Gets the lists across a client to which a subscriber with a particular email address belongs.
  • get_people() — returns the people associated with this client.
  • get_primary_contact() — retrieves the email address of the primary contact for this client
  • get_scheduled() — Gets a list of scheduled campaigns for the current client
  • get_segments() — Gets all list segments the current client has created
  • get_suppressionlist() — Gets all email addresses on the current client's suppression list
  • get_templates() — Gets all templates the current client has access to
  • set_basics() — Updates the basic information for a client
  • set_client_id() — Change the client id used for calls after construction
  • set_monthly_billing() — Updates the billing details of the current client, setting the client to the monthly billing model For clients not set to pay themselves then the markup percentage field is ignored
  • set_payg_billing() — Updates the billing details of the current client, setting the client to the payg billing model For clients not set to pay themselves then all fields below ClientPays are ignored All Markup fields are optional
  • set_primary_contact() — assigns the primary contact for this client to the person with the specified email address
  • suppress() — Adds email addresses to a client's suppression list.
  • transfer_credits() — Transfer credits to or from this client.
  • unsuppress() — Unsuppresses an email address by removing it from the the client's suppression list.

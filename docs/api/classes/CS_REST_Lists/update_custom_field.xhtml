CS_REST_Lists::update_custom_field

Updates a custom field for the current list

    Signature

    public function update_custom_field(string $key, array $custom_field_details )

    Parameters

    $key — string
    The personalisation tag of the field to update
    $custom_field_details — array
    The details of the new custom field.
    This array should be of the form
    array(
    'FieldName' => string The new name for the field
    'VisibleInPreferenceCenter' => boolean representing whether or
    not the field should be visible in the subscriber preference
    center.
    )

    Returns

    CS_REST_Wrapper_Result
    A successful response will be the
    personalisation tag of the updated custom field

CS_REST_Lists::get_custom_fields

Gets a list of all custom fields defined for the current list

    Signature

    public function get_custom_fields()

    Returns

    CS_REST_Wrapper_Result
    A successful response will be an object of the form
    array( { 'FieldName' => The name of the custom field 'Key' => The personalisation tag of the custom field 'DataType' => The data type of the custom field 'FieldOptions' => Valid options for a multi-optioned custom field 'VisibleInPreferenceCenter' => Boolean representing whether or not the field is visible in the subscriber preference center } )

CS_REST_People::update

Updates details for an existing person associated with the specified client.

    Signature

    public function update(string $email, array $person )

    Parameters

    $email — string
    The email address of the person to be updated
    $person — array
    The updated person details to use for the update.
    This array should be of the form
    array (
    'EmailAddress' => The new email address
    'Name' => The name of the person
    'AccessLevel' => the access level of the person
    'Password' => (optional) if specified, changes the password to the specified value
    )

    Returns

    CS_REST_Wrapper_Result
    A successful response will be empty

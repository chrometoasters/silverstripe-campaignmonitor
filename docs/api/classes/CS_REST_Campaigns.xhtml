\CS_REST_Campaigns

Class to access a campaigns resources from the create send API.

This class includes functions to create and send campaigns,
along with accessing lists of campaign specific resources i.e reporting statistics

  • Author: tobyb

Methods

public

  • CS_REST_Campaigns() — Constructor.
  • create() — Creates a new campaign based on the provided campaign info.
  • create_from_template() — Creates a new campaign from a template based on the info provided.
  • delete() — Deletes an existing campaign from the system
  • get_bounces() — Gets all bounces recorded for a campaign
  • get_clicks() — Gets all clicks recorded for a campaign since the provided date
  • get_email_client_usage() — Gets the email clients that subscribers used to open the campaign
  • get_lists_and_segments() — Gets the lists a campaign was sent to
  • get_opens() — Gets all opens recorded for a campaign since the provided date
  • get_recipients() — Gets all email addresses on the current clients suppression list
  • get_spam() — Gets all spam complaints recorded for a campaign since the provided date
  • get_summary() — Gets a summary of all campaign reporting statistics
  • get_unsubscribes() — Gets all unsubscribes recorded for a campaign since the provided date
  • send() — Sends an existing campaign based on the scheduling information provided
  • send_preview() — Sends a preview of an existing campaign to the specified recipients.
  • set_campaign_id() — Change the campaign id used for calls after construction
  • unschedule() — Unschedules the campaign, moving it back into the drafts. If the campaign has been sent or is in the process of sending, this api request will fail.

CS_REST_General::exchange_token

Exchange a provided OAuth code for an OAuth access token, 'expires in' value and refresh token.

    Signature

    public function exchange_token(int $client_id, string $client_secret, string $redirect_uri, string $code )

    Parameters

    $client_id — int
    The Client ID of your registered OAuth application.
    $client_secret — string
    The Client Secret of your registered OAuth application.
    $redirect_uri — string
    The Redirect URI of your registered OAuth application.
    $code — string
    The unique OAuth code to be exchanged for an access token.

    Returns

    CS_REST_Wrapper_Result
    A successful response will be an object of the form
    { 'access_token' => The access token to use for API calls 'expires_in' => The number of seconds until this access token expires 'refresh_token' => The refresh token to refresh the access token once it expires }

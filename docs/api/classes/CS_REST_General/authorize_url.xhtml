CS_REST_General::authorize_url

Get the authorization URL for your application, given the application's Client ID, Client Secret, Redirect URI, Scope, and optional state data.

    Signature

    public function authorize_url(int $client_id, [string $redirect_uri = NULL, [string $scope = NULL, [string $state = NULL]]] )

    Parameters

    $client_id — int
    The Client ID of your registered OAuth application.
    $redirect_uri — string
    The Redirect URI of your registered OAuth application.
    $scope — string
    The comma-separated permission scope your application requires.
    See http://www.campaignmonitor.com/api/getting-started/#authenticating_with_oauth for details.
    $state — string
    Optional state data to be included in the URL.

    Returns

    string
    The authorization URL to which users of your application should be redirected.

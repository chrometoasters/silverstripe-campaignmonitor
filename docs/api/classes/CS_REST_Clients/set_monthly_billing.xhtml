CS_REST_Clients::set_monthly_billing

Updates the billing details of the current client, setting the client to the monthly billing model For clients not set to pay themselves then the markup percentage field is ignored

    Signature

    public function set_monthly_billing(array $client_billing )

    Parameters

    $client_billing — array
    Payg billing details of the client.
    This should be an array of the form
    array(
    'Currency' => The currency fees are paid in
    'ClientPays' => Whether client client pays for themselves
    'MarkupPercentage' => Sets the percentage markup used for all monthly tiers
    )

    Returns

    CS_REST_Wrapper_Result
    A successful response will be empty

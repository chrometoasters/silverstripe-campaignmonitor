CS_REST_Clients::get_campaigns

Gets a list of sent campaigns for the current client

    Signature

    public function get_campaigns()

    Returns

    CS_REST_Wrapper_Result
    A successful response will be an object of the form
    array( { 'WebVersionURL' => The web version url of the campaign 'WebVersionTextURL' => The web version url of the text version of the campaign 'CampaignID' => The id of the campaign 'Subject' => The campaign subject 'Name' => The name of the campaign 'FromName' => The from name for the campaign 'FromEmail' => The from email address for the campaign 'ReplyTo' => The reply to email address for the campaign 'SentDate' => The sent data of the campaign 'TotalRecipients' => The number of recipients of the campaign } )

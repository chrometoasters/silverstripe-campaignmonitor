CS_REST_Clients::get_scheduled

Gets a list of scheduled campaigns for the current client

    Signature

    public function get_scheduled()

    Returns

    CS_REST_Wrapper_Result
    A successful response will be an object of the form
    array( { 'CampaignID' => The id of the campaign 'Name' => The name of the campaign 'Subject' => The subject of the campaign 'FromName' => The from name for the campaign 'FromEmail' => The from email address for the campaign 'ReplyTo' => The reply to email address for the campaign 'DateCreated' => The date the campaign was created 'PreviewURL' => The preview url of the campaign 'PreviewTextURL' => The preview url of the text version of the campaign 'DateScheduled' => The date the campaign is scheduled to be sent 'ScheduledTimeZone' => The time zone in which the campaign is scheduled to be sent at 'DateScheduled' } )

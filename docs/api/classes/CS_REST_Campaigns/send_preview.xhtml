CS_REST_Campaigns::send_preview

Sends a preview of an existing campaign to the specified recipients.

    Signature

    public function send_preview(array<string> $recipients, [string $personalize = 'Random'] )

    Parameters

    $recipients — object
    The recipients to send the preview to.


    $personalize — string
    How to personalize the campaign content. Valid options are:
    'Random': Choose a random campaign recipient and use their personalisation data
    'Fallback': Use the fallback terms specified in the campaign content

    Returns

    CS_REST_Wrapper_Result
    A successful response will be empty
